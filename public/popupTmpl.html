Help

Help contents for Outlink Tool.

Normal Search Mode

The normal search mode will allow users to search by both URL's and keywords. It will return all results that contain the search term.

For example searching for the term 'Undergraduate' will return results like:

  • http://www.uel.ac.uk/Undergraduate/Art-and-Design
  • http://www.uel.ac.uk/Undergraduate/Mechanical-Engineering
  • http://www.uel.ac.uk/Undergraduate/Mechanical-Engineering
Exact Match Search Mode

The 'exact match' search mode will allow the user to search by an exact URL. Only exact matches will be returned in this mode.

For example searching for http://www.uel.ac.uk/Undergraduate/Art-and-Design will return only the Page url results that match that url exactly.

Show Entries Field

This should be used to control the number of results that are loaded onto the page at one time.

*Please note that setting this to a significantly high number may cause performance issues.

Load More Results Link

The 'Load More Results' link can be used to load more results onto the page. The amount of results that are loaded is determined by the number set in the 'show entries' field.

Clicking the 'load more' link repeatedly will load the data in increments of the number specified in the 'show entries' field. The data will continue to be appended to the end of the page until the total number of results are showing on the page.

This method of displaying search results has been implemented to allow the browser to handle hundreds of thousands or even millions of results without resulting in performance/termination issues.

Download to CSV

This link will allow the user to download the entirety of the search results to a csv file, which can be viewed in a Spreadsheet.
